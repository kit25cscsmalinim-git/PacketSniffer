# Packet Sniffer using Scapy

## 📌 Project Title
Network Packet Sniffer using Python (Scapy)

## 📖 Description
This project captures and analyzes network packets in real-time using Python and the Scapy library. It extracts important details such as source IP, destination IP, protocol type (TCP/UDP), and timestamp.

All captured packet information is displayed on the console and stored in a text file (`packets.txt`) for further analysis.

## 🌍 SDG Mapping
This project maps with **SDG 9: Industry, Innovation and Infrastructure**.  
It helps in building secure and efficient network infrastructure through traffic monitoring and basic analysis.

## ⚙️ Tools & Technologies Used
- Python  
- Scapy Library  
- Networking Protocols (IP, TCP, UDP)  

## 🚀 Features
- Real-time packet capturing  
- Displays packet number, time, protocol, and IP addresses  
- Supports TCP and UDP detection  
- Logs packet data into a file (`packets.txt`)  
- Simple and lightweight implementation  

## 🧠 Working Principle
The program uses Scapy's `sniff()` function to capture packets.  
Each packet is processed to extract:
- Source IP  
- Destination IP  
- Protocol type  
- Timestamp  

The data is printed and saved for monitoring purposes.
